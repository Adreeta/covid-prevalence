COVID-19 Point-Prevalence Estimation Map

DRAFT

Last updated 2020-June-28

**NOTE** This data is not currently being updated. This website is for proof-of-concept and demonstration only. The system is currently being configured to run this product with at least daily updates.

This map shows the estimated point-prevalence over time of COVID-19. The point prevalence is estimated using Bayesian inferrence based on the latest epidemiological parameters and open-souce case data. See https://gitlab.com/stevenhorn/covid-prevalence for more information.
